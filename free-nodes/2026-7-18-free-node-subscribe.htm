第一章 Shadowrocket R的核心价值解析

1.1 什么是Shadowrocket R？

Shadowrocket R是一款专为iOS平台开发的高级网络代理客户端，它通过智能路由和多重加密技术，帮助用户突破网络限制，实现全球内容的无障碍访问。不同于传统VPN的单一功能，Shadowrocket R集成了多种代理协议和规则管理功能，堪称移动端代理工具的"瑞士军刀"。

1.2 六大核心功能亮点

  1. 全协议支持：
    支持HTTP/HTTPS/SOCKS5/SS/SSR/Vmess/Trojan等主流代理协议，几乎覆盖所有代理服务类型。

  2. 智能分流系统：
    独创的规则引擎可实现国内外流量智能分流，国内直连提升速度，国外走代理确保访问。

  3. 极简美学界面：
    采用iOS原生设计语言，功能布局符合人体工学，状态指示灯、流量统计等关键信息一目了然。

  4. 云端配置同步：
    支持iCloud同步服务器配置，更换设备时无需重复设置，真正做到"一次配置，处处可用"。

  5. 高级网络诊断：
    内置Ping测试、Traceroute等专业工具，连接问题快速定位。

  6. 隐私保护机制：
    严格的无日志政策配合AES-256加密，确保网络活动不被监控和记录。

第二章 从零开始的完整配置指南

2.1 获取与安装

iOS用户专属步骤：
1. 进入App Store搜索"Shadowrocket"（注意：部分地区可能需要切换商店区域）

重要提示：

- 安装前建议关闭其他VPN应用
- 如遇下载失败，可尝试切换WiFi/4G网络

2.2 服务器配置详解

手动添加节点：
1. 点击右下角"+"按钮
2. 选择对应协议类型（SS/SSR/Vmess等）
3. 填写：
- 服务器地址（支持域名/IP）
- 端口号（通常为443或自定义端口）
- 加密方式（推荐AES-256-GCM或ChaCha20）
- 密码/UUID（Trojan协议专用）

订阅链接自动更新：
1. 进入"订阅"标签页
2. 粘贴服务商提供的订阅URL
3. 设置自动更新频率（建议6-12小时）

2.3 规则配置的艺术

基础规则设置：
- GEOIP规则：自动识别国内外IP段
- 域名后缀规则：指定特定域名走代理
- 关键词规则：匹配URL中的特定字符

第三章 实战问题解决方案库

3.1 连接类问题

症状：图标显示已连接但无法访问
- 检查节点有效期/流量余额
- 尝试切换TCP/UDP传输模式
- 关闭IPv6支持（设置→高级）

症状：频繁断线
- 调整MTU值（建议1400-1500）
- 启用"连接保持"功能
- 更换备用端口（如443→8443）

3.2 速度优化方案

网络调优三要素：
1. 节点选择：优先选择物理距离近、负载低的服务器
2. 协议优化：移动网络推荐使用SS+AEAD加密，宽带可尝试Vmess+WS+TLS
3. 参数调整：适当增大缓冲区大小（Advanced→Buffer Size）

3.3 隐私保护进阶

防DNS泄漏设置：
- 启用"严格路由"模式
- 自定义DNS服务器（推荐1.1.1.1或8.8.4.4）
- 开启"伪装类型"（针对V2Ray协议）

第四章 专家级使用技巧

4.1 多设备协同方案

家庭共享配置：
1. 主设备完成配置后导出为.ssr/.vmess文件
2. 通过AirDrop或iMessage分享给家庭成员
3. 其他设备导入时注意修改本地端口避免冲突

4.2 企业级应用场景

远程办公解决方案：
- 配置公司内网专用节点
- 设置分流规则使办公系统直连
- 结合Shortcuts实现自动化连接

4.3 极限网络环境应对

卫星网络/高延迟环境：
- 启用"压缩传输"选项
- 调整协议为UDP-only
- 降低加密强度换取速度（如改用RC4-MD5）

FAQ

如何将 Windows 上的 V2Ray 与 WSL 环境联合调试？
在 WSL 内运行需要的调试工具并通过 localhost 或将 V2Ray 监听地址绑定为 0.0.0.0，然后在 Windows 与 WSL 之间配置端口映射。注意 Windows 防火墙规则需允许 WSL 与 Windows 进程通信。
QuantumultX 如何实现按端口和协议分流？
QuantumultX 支持按 TCP、UDP、端口及应用协议分流流量，结合策略组选择最优节点。可为游戏、视频和广告屏蔽提供最佳网络体验，提高访问速度和稳定性。
V2Ray 的 WebSocket + TLS 如何防止流量被检测？
WebSocket + TLS 可将 V2Ray 流量伪装成正常 HTTPS 请求，隐藏真实数据特征，配合域名伪装和策略组使用，可有效绕过封锁，提高节点安全性和连接稳定性。
Surfboard 策略组如何提升游戏和视频体验？
通过策略组将游戏、视频和浏览器流量分配到低延迟、高带宽节点，Surfboard 可自动切换最优节点，减少延迟和卡顿，提高整体网络体验和访问速度。
V2Ray 支持哪些传输层协议？
V2Ray 支持多种传输层，包括 TCP、mKCP、WebSocket、gRPC、QUIC 等。每种传输方式都有不同的性能特点，例如 TCP 稳定、WS 兼容 CDN、QUIC 延迟低。
VMess 使用 AEAD 加密的优势是什么？
AEAD 加密提供更强的安全性和抗重放攻击能力，同时隐藏流量特征，降低被 DPI 检测风险。在高封锁环境下使用，可提升连接稳定性和长期可用性。
PassWall2 的延迟测速功能如何提高网络体验？
PassWall2 可以定期测速节点延迟，并结合策略组自动选择最优节点。这样在访问网站或视频时，可以始终使用延迟最低的线路，提高网络稳定性和速度。
如何合理设置 Clash 的 proxy-group 实现自动切换？
创建包含多个节点的策略组（如 `url-test` 或 `fallback`），设置合适的检测 URL 与检测频率，启用延迟优先或可用性优先策略。这样客户端会自动选择延迟最低或可用节点，提升稳定性。
Nekoray 与 Nekobox 有什么不同？
Nekoray 是一款基于 Qt 的跨平台代理前端，而 Nekobox 更偏向移动端简化使用。两者均支持 V2Ray/VLESS/Trojan 协议，但 Nekobox 提供了更轻量的界面和更快的连接启动速度。
Nekobox 能否同时管理多个客户端节点？
支持。Nekobox 与 Shadowsocks、Clash 等客户端独立运行，互不干扰。用户可通过策略组灵活管理节点，实现多客户端分流和网络优化，提高使用便利性。
